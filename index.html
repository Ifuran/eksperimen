Meta Tag
Meta Tag adalah tag html untuk mendeskripsikan halaman web kita pada mesin pencari. Tag ini tidak akan mempengaruhi isi dari halaman kita. Detail.
Meta Tag digunakan agar mesin pencari menampilkan halaman web yang sesuai dengan pencarian pengguna. Umumnya Meta Tag diletakkan di bagian header HTML.
Meta Tag Title
Meta Tag ini menjelaskan judul halaman kita ke mesin pencari. Detail.
Penulisannya: <title>judulhalaman</title>
Meta Tag Description
Meta Tag ini menjelaskan deskripsi halaman web kita ke mesin pencari. Detail.
Penulisannya: <meta content="deskripsihalaman" name="Description">
Meta Tag Keyword
Meta Tag ini menjelaskan keyword halaman kita ke mesin pencari. Detail.
Penulisannya: <meta content="keywordhalaman" name="Keyword">
Meta Tag Charset
Meta Tag ini menjelaskan karakter atau penulisan huruf yang kita gunakan di halaman ini pada mesin pencari. Detail.
Penulisannya: <meta charset="utf-8">
